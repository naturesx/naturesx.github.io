我的硬件代号

电脑A（笔记本）
这个是2016年高考结束后购买的
宏碁E5-572G 当时买的3500 算是低端小钢炮吧
陪伴了我很多年 虽然磕磕绊绊 但没有什么大问题发生过
再加上这个价位 我已经满意了
电脑B（台式）
这个是我在高中 找哥们装的电脑
主机 显示器 键鼠 音响啥都算上 4000左右
这个机器陪伴我的时间就更长了
但是我高中毕业后 平时也不经常在家
家里也没人用 所以使用频率算是比较低了
假期我一般会用他打打游戏
电脑C（服务器）
这个是我研究生阶段 实验室的服务器 linux系统
当然不止我一个人使用
电脑D (苹果)
正在考虑

创建个人博客历程

##Hexo

先去nodeJS官网下载NODEJS
在命令行安装cnpm（淘宝的源） hexo
当进行到hexo init时，需要git clone
然而电脑A的git clone一直有问题
username和email啥的都设置过
ssh密钥也在github官网上设置过了
还重装过git …依然无效果
使用gitclone会报错 memory oversize 88…… bytes（具体记不清了）
尝试鼓捣一段时间后 只能放弃

之后在电脑B（还是win10）上再试了一次 就成功了
然而 电脑B的GIT连user和email啥的都没设置过
让人摸不到头脑 或许电脑A重装下系统就好了（等我毕业）

##hugo

这个没怎么研究过
一个是因为语言载体是GO语言 之前没接触过
二一个是 安装文件没下载成功
我之后会在电脑A上试试看（等待后续更新）

##docker

这个感觉更适合linux/MACOS
windows上虽然也能用 但是实现方式是在WIN上安装个虚拟机 再进行docker的安装
然而 我没有找到教程中所提到的WIN10的微软自带模拟机（而且开启有副作用）
敲命令也没能开启 所以就先这样吧
详情见菜鸟教程的网站 搜docker即可

##VUE全家桶+nodeJS
我会在电脑A上先试一下这个（等待后续更新）
